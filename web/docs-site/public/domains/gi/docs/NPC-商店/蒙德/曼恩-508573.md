# 曼恩

## 基本信息

**位置**: 荆夫港
**性别**: 男

## 对话内容

**【开场白】**

> 上山，上山…就是上了山，也找不着疗养院咯…

> 早就拆咯。

> 哪怕没有什么闹鬼的事，你上去也找不着什么宝藏，还是早点回家吧。

> 我在那儿住了整整七年。山上的寒冷仿佛能连时间与风一块儿封住，七年就跟七天似的，一眨眼就过去咯。

> 但人总是要下山的。哪怕被风推着走，也是在往前走。

> 没什么可看的，也没什么可找的。要是上去以后对着一片废墟大失所望，可别怨老头子我没提醒过你。
