# 小鱼【「小鱼大王」】

## 基本信息

**位置**: 荆夫港

## 对话内容

**【开场白】**

> …呜汪？

> 欸？旅行者你看，这条小狗好像很饿的样子…

> 唔，看起来也不像是流浪狗…是和主人走丢了吗？

> 呜...

> 总觉得不忍心放着不管，饿肚子的感觉最难受了…旅行者，要是我们有多余的肉，分一块给它吧？

> 呜汪…！

> 跟过来了呢…唔，就这样把它带走也不是办法，我们在附近找找看它的主人吧？

> 小、小鱼！太好了，终于找到了！呜呜，我还以为再也见不到你了…！

> 呜汪！

> 谢谢你们，好心的大哥哥/姐姐们！
