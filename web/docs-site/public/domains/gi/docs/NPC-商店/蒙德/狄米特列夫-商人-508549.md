# 狄米特列夫【商人】

## 基本信息

**位置**: 荆夫港
**性别**: 男

## 对话内容

**【开场白】**

> 马上…他要是来了，我就得救了。

> 唉，我在等埃斯特拉冈。他说要来我这取货，可是到现在都没有来…已经不知道迟到多久了。

> 他要是不来，我就只能在这里等他。没有办法，只能如此。

> 对，我们约定在这个地方见面的。可是他一直不见踪影。我也不敢抛下货物去找他，只得在此等待。

> 我在这里日复一日地等待他，他要是来了，我就得救啦。

> 我简直等得想要上吊了。
