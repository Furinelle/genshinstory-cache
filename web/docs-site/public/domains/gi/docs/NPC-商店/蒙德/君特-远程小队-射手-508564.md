# 君特【「远程小队」射手】

## 基本信息

**位置**: 荆夫港
**性别**: 男

## 对话内容

**【开场白】**

> 还得是剑好养啊…洛恩的玩笑也有道理，铳枪维护起来实在麻烦…

> 那倒不是，这玩意可好用了。如果你准头不错，甚至有机会在魔物毫无防备的情况下，把它们远远拿下。

> 只不过，开枪爽归爽，保养真太难…

> 就算使用了最新的无烟火药，发射时的污垢大大减少，每天都必须用通条好好把枪管和枪膛清理一番。

> 上枪油的时候，一个不小心，就会弄得满手都是…每打三百七十发左右，都必须把整支统枪拆解，仔细清洗擦拭。

> 要是有支援小队的枪匠们在，这一切倒还好说。

> 偏偏我们远程小队的部署模式很不一样，通常被拆分成多个小组，加强到其他小队，作为强化战力的一部分…

> 枪匠们不可能一直跟着我们，如果战斗强度比较高，每天都得拆枪清洁。

> 你是说…像愚人众先遣队的那些射手，发射元素进行攻击？唉，那可高看我们了。

> 虽然我们使用的统枪，的确结合了至冬、枫丹与蒙德的三重技术…但想要达成愚人众射手们的效果，还缺少了某些核心部件。

> 普通人想要操纵元素力，就像让松鼠学会怎么用「旗舰」的点唱机。

> 除了运用偏门的方法，将含有元素的成分封装在容器里，整个投掷出去…以我有限的见闻，暂时不知道该怎么办。
