# 莱纳斯【吟游诗人】

## 基本信息

**位置**: 荆夫港
**性别**: 男

## 对话内容

**【开场白】**

> 孤独的帆儿闪着白光…

> 呵呵，没错。不知道您有什么想听的曲子呢？

> 无论是早春荒地的丁香、轻眠在冻土中的希望，还是夏日绽放的繁花、暖风中的欢歌，都是经久不衰的诗题。让我选一首…

> 「…五月天，无吝悭，赐欢颜，以此祛忧烦。春光复归，原野间，信可见，花争艳…」

> 「…蛰虫始振，百鸟争鸣，林峦尽欢声。唯我忆暮冬之寒，歌阒然…」

> 您的香氛…真是特别呢。每种香气的平衡，把握得堪称绝妙，就像是美妙的协奏曲…
