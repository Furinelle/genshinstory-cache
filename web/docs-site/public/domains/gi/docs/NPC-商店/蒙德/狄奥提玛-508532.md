# 狄奥提玛

## 基本信息

**位置**: 大方广之间·图书馆三层
**性别**: 女

## 对话内容

**【开场白】**

> 你是新来的吗？别紧张，你能听懂我说话，说明「被提」的过程没出什么问题。

> 我是来自梅洛皮斯的狄奥提玛，此处最早的一批住民。在我来的时候，这里还没有什么藏书，只有一片寂寥的荒野。

> 那时我还是一名船工，负责将流离的灵魂送上银璨璨的大船，航向云上的月都。

> 说起来和我现在做的也差不多。

> 嗯，现在的人都不知道了吧，在我的时代天上还挂着三轮明月，每一轮月亮上都住着许多月使。

> 那时高贵的灵魂不是前往古树的深处，而是跟随月使的行迹，在月上的城市永眠。

> 虽然我没去月亮上看过，不过想来应该是个和这里一样安逸的地方吧。
