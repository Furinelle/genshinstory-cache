# 长孙继璟

## 基本信息

**位置**: 麓阳书院
**性别**: 男

## 对话内容

**【开场白】**

> 此处云气虽盛，却为何迟迟不降甘霖…

> 我在等雨。

> 在下长孙继璟，人称「凝雨剑」。想当年，恩师传我古华团绝学「裁雨」——剑承凝雨，势法飘华…

> 依在下所观，正是要将雨凝于剑上，因此必须要在雨天修习。

> 只可惜此地万世如一，虽有流云，却从未见雨点落下。

> 没有雨，我要如何裁？岁月悠悠，我的剑术竟是寸步难行…
