# 薇恩狄娅

## 基本信息

**位置**: 大方广之间·图书馆二层
**性别**: 女

## 对话内容

**【开场白】**

> 小心脚下呀，亲爱的。这地方到处都是书，要是不小心掉进哪本奇怪的书里，可就难办啦。

> 我的名字是薇恩杜洁妮妲·德·乌塞洛布罗吉。

> 很难念吧？这名字长得连我自己都懒得念。所以您直接像那些雷穆利亚人一样，管我叫薇恩狄娅就好。

> 卡斯蒂廖内先生之前说，后世的高海好像管我叫什么「雪翅雁夫人」…不过您没必要这么叫，听上去像是个奇怪的老太婆。

> 我过去曾经是乌塞洛布罗吉的女王，至于现在嘛，就只是个随处可见的童话作家而已。

> 其实本来也没什么区别，我当女王的时候，也不过是负责微笑和点头。无事可做，我就只能偷偷写点童话故事。

> 无论是向雷穆利亚人投降，还是向埃雷莫利卡的高王投降，都是摄政的长老们决定的事。

> 所以，您看，哪怕我被那位神明带到这里来，也根本没人在意。

> 况且，这样一来，我也不用被迫去埃雷莫利卡觐见、赔笑，听那些甚至不洗澡的野蛮人…

> 抱歉，我是说我可敬的同胞们。总之，我不用再听他们和那个活该一辈子嫁不出去的女人为了要杀多少人争论不休了。多好呀。
