# 玛卡尼哈冈

## 基本信息

**位置**: 「沙上楼阁」
**性别**: 男

## 对话内容

**【开场白】**

> 孤乃统领七域的大哈冈，赤土之主玛卡尼大王，今日得以觐见孤是你的荣幸。

> 什么？难道你从没有在刻在方尖碑上的铭文中读到过孤的名号吗？该不会在历史上留下姓名的，反而是孤的手下败将呼玛云吧？

> 哼，我早该知道我豢养的那些个诗人和书记官都是骗吃骗喝的饭桶，当初就应该把他们全部处死。

> 可惜你没见过孤麾下那如黑云般的军势，赤霞般的旌旗，还有那些被孤的马蹄踏碎的王座。

> 没想到那样的霸业也不过是过眼云烟，被世人所遗忘，不像图莱杜拉的那些文弱之主，反而有人给他们歌功颂德。
