# 狩野

## 基本信息

**位置**: 麓阳书院
**性别**: 男

## 对话内容

**【开场白】**

> 哦哦，是旅行者啊！

> 我正在这里采风，希望能捕捉到群鹤聚集的场景，绘成《白鹤竹石四时八景图》。

> 只是白鹤不落，便唯有待至其落。

> 唔，只是等着的话要好久吧？既然白鹤不落，那我们就想办法引诱它们过来好啦。

> 嘿嘿，这样一来，鹤就全部被吸引过来啦！

> 竟是用这般技法解决的…何等奇绝。

> 哼哼，只是等在原地的话可是没办法解决问题的！——怎么样，我是不是说了句听上去很有道理的话？

> 多谢旅行者和派蒙。我改主意了，只画鹤太俗套了，就和画屏风一样。

> 这次的作品，我要改画《旅行者派蒙引鹤图》了。灵感来了，请收下这些薄礼。

> 不，说到底不还是要画鹤嘛！
