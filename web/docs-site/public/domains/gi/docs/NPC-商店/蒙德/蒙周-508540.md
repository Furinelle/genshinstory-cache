# 蒙周

## 基本信息

**位置**: 大方广之间·图书馆一层
**性别**: 男

## 对话内容

**【开场白】**

> 大道废，有文章♪~智慧出，多书蠹♪~

> 怎么了，因为这里是读书的地方？正因如此，我才要唱歌呢！

> 不管多么荒诞无稽的话，只要写成文字，就会被当成事实。这里到处都是这样可笑的书。

> 在我们琅玕国，从来没人会写这种东西。不管遇到什么事情，人们都只会把它们编成歌谣。

> 大王派下来的采诗官听到这些歌谣后，就会记下来，回到宫廷里唱给那些大人们听。

> 这里头饱含着的真情，可比那些破文章有意思多了。

> 谁知道呢。兴许是哪个自大的家伙，觉得把什么「文明」「历史」保存起来是件了不得的大事。

> 于是就连我也被当作「历史」的一部分塞了进来。
