# 托萨利斯

## 基本信息

**位置**: 大方广之间·图书馆三层
**性别**: 男

## 对话内容

**【开场白】**

> 呜哇，吓我一跳…还没看到更后面呢，没想到还会有怪物从书里跳出来…

> 啊…我曾是语言，如今却沦为了错觉破碎的主体，再度回归这不完美的「象征」。

> 呜哇！奇怪的家伙说着奇怪的话从奇怪的书里出现了！

> 「奇怪」是一种不严谨的表达。它意味着无法被描述、无法被书写、无法被表征的特殊性主体，而我显然不符合这个定义。

> 我不过是试图在书页中寻找通往「实在」的入口，却不慎滑入了无限延宕的隐喻中…

> 我成为了无法被象征化的剩余，多亏你的介入，我才得以从语言的暴政中被剥离，可以重新开始误认我自己了。

> …我不小心掉进书页里出不来了，谢谢你把我拉出来。

> 啊…一个「偶然性」的「具象」，我看到它正在走出「此在」的「沉沦」。

> 很好，你很敏锐，抓住了「偶然性」这个词。

> 「偶然性」意味着无法预知的未来与可能，而拥有可能便等同于拥有希望。

> 当初那位圣徒就是这么对我们说的。但是很可惜，在我来到这里后不久，他所拥有的可能性就灰飞烟灭了。

> 的确是个悲剧…不过这就是历史。历史和语言一样是流动的，就像词汇也没有固定的所指。

> 所以词汇本身并不重要，重要的是我们已经达成了理解，完全可以说这是一场成功的对话。
