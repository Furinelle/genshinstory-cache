# 加奥凯

## 基本信息

**位置**: 荆夫港
**性别**: 男

## 对话内容

**【开场白】**

> 哈哈，没想到这里居然有这么完美的跳台！

> 不错！身为梅兹特利的战士，就该像图帕克、柯夏妮娜与乌努库那样，时刻磨砺武技，哪怕是养伤的时候也——

> 等、等等！这位先生！您先冷静下来，有什么话慢慢说…

> 从这么高的地方跳下去，就算是枫丹的职业潜水员，也难免要受伤的！

> 嗯？枫丹？

> 哈哈，那就对了！想当年，大英雄乌努库的恋人正是来自枫丹！他也正是为了与心上人重逢，才会投海——

> 投、投海？！不行！我绝对不会让你寻短见的！

> 没错！小妹，别低估我们梅兹特利的水性了——呃，等等，别拽我啊！
