# 卡斯蒂廖内

## 基本信息

**位置**: 麓阳书院
**性别**: 男

## 对话内容

**【开场白】**

> 狩野啊，你屏风还没画完就别到处乱窜了，先想想你的瀑布怎么画吧。看在厄歌莉娅的份上，哪有人用散点透视画瀑布的…

> …嗯？啊，抱歉，原来是陌生的朋友啊。听脚步声我还以为是狩野呢。

> 如您所见，我在作画。

> 为了避免您误会，请让我先和您说清楚。虽然我擅长画动物，但我画的动物不是用来吃的。

> 欸，原来能吃吗？！

> 画中人是眼前人，画中兽自然也能是盘中兽…唉，虽说如此，还是希望您不要像璃月的那位妌癸女士一样，只缠着我画鱼肉。

> 艺术——可敬的艺术！无论时代，其主旨应当是人类灵魂的救赎、情操的陶冶，以真、善、美指引我等，通往更高的精神境界。

> 举例来说，登临山峰的最高处，所见的风景自然真实无阻。我所追求的艺术也是如此。

> 嗯…只是像这样说的话，您也许没有办法直观理解。这样吧，我给您画个浮台，您上去体验一番便是。

> 不是不是，您误会了，我是来自佩特莉可镇的枫丹人。

> 只不过，既然来到了这如同仙境的宝地，自然要换上璃月雅士的衣冠，方能对得起这满眼的如画江山。
