# 杨轸【忧伤的商人】

## 基本信息

**位置**: 荆夫港
**性别**: 男

## 对话内容

**【开场白】**

> 好想吃万民堂的烤吃虎鱼啊…

> 看您的装束…应该是位旅行者吧。不知道您有没有去过万民堂…

> 实不相瞒，我实在是想念家乡的口味。唉，可惜啊，我目前回不去喽。

> 唉…说来惭愧，我是璃月的茶叶商人，平日里做些海运的生意。

> 未曾想，先前在海上遭了风暴，货物损失大半…待运到荆夫港的时候，连成本都没能收回来。

> 我不想就这么灰溜溜地回去，所以想在这地方先做些别的买卖，至少先赚回本钱。只是…

> 罢了。不过是无用的牢骚，我也不过是馋烤吃虎鱼罢了。让您见笑了。
