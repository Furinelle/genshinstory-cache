# 虞黄衣

## 基本信息

**位置**: 大方广之间·第四层
**性别**: 男

## 对话内容

**【开场白】**

> 「待到你一脚踏碎混沌窍，方向是道…」

> 我的时代，似乎也没有什么特别的。

> 当时大家都围着天衡山住，有力气的锤金打铁。身手好的去山上猎肉采菇。

> 摩拉克斯——哈，那时既不是帝君，也不是岩王——他自称莫先生。

> 「神器未失，风云不起。四方泰平，且居这浮世一隅。」

> 我听人说了后来的事情。天下大乱，莫先生的「一隅」变大了不少啊。
