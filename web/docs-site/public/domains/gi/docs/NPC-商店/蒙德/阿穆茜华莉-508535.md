# 阿穆茜华莉

## 基本信息

**位置**: 柱殿中央区
**性别**: 女

## 对话内容

**【开场白】**

> 可憎的守护者，我不过是稍稍离开一会儿，就把这地方给折叠起来了…

> 果然，圣王陛下说得对，像这样超越人类力量的存在，就应当被烈阳焚尽…

> 赞美烈阳。愿四方的炎风和火雨焚尽你的仇敌，尊贵的朋友。

> 我是阿穆茜华莉，诞生于烈阳归火后的第一个七芦苇之年，曾为至仁至耀的圣王担任审录官。

> 我原本正在搜捕一名背弃联盟誓言、忘却烈阳教诲的叛徒，不料却被困在了此处。

> 我的祖先曾与那位真正的神明同饮，而今日，作为烈阳在卑微的大地上投下的影子，我在此向你问候。

> 烈阳赐福了我们伟大的联盟，要我们用焚火彻底烧尽一切不洁之物，也就是一切阻碍人类掌握自身命运的邪祟之物。

> 我知晓后世的叛徒们灭亡了联盟，将凡人视为新的太阳，还将圣火的看护者污蔑为暴君。我不会与他们争辩。

> 因为烈阳从未死去。终有一日，真正的神明会重新归来，烧却这个污秽纪元的大地，而困在这里的所有人，都将迎来希望中的新生。
