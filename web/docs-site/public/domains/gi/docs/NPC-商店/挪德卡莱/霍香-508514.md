# 霍香

## 基本信息

**位置**: 死兆星级二号舰
**性别**: 女

## 对话内容

**【开场白】**

> 你好啊，旅行者。「水面战与陆攻支援负责员」，向你报到。

> 顾名思义，北斗大姐头交给我的任务有两个，其一是水面作战，指挥船员们打击海上的魔物与贼匪们的船只。

> 其二是控制改进型的归终机，按信号发射指定弹药。对空拦截时还要选择不同颜色的烟花示踪剂，以便进行毁伤评估。

> 是的，你在战斗区域发出信号后，由我这边从船上打出弹药。

> 而你照射目标所用的俯瞰机关，本身也由一发先导弹药投送到战区上空，所以需要准备时间，留在空中的窗口也不长…

> 当然，目前我们这种配合模式，仅限于单舰作战的情景。毕竟我们「南十字」船队不需要击溃乌泱泱的魔物大军。

> 听说，碰见强度极高的任务时，千岩军会拿出「足以持续盘旋一天」的俯瞰机关，不断用远程火力压制对方…

> 我反正想象不出来，什么样的对手需要用如此饱和的火力去狠狠打击…还是别开这个眼界为妙。
