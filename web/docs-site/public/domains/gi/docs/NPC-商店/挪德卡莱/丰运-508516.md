# 丰运

## 基本信息

**位置**: 死兆星级二号舰
**性别**: 男

## 对话内容

**【开场白】**

> 罐头、干货和冻品还足够，但船都来近海了，还是去买些新鲜食材吧…

> 哦哦，我不掌勺啦，我只负责这艘船的后勤和仓储，大厨正在船舱里忙活呢。

> 哎呀，船上的食材为了长期航行，都做了脱水和冷冻处理，要么就是罐装货，口感不太好，不然，我高低让他给你炒俩菜！

> 严格来说也不算差啦…

> 听说啊，以前的水手出港后，每天的菜谱都是定死的，先吃容易腐烂的蔬菜水果，按时序往后一天一天排…

> 最后就只剩下硬得能磕碎牙的肉干，和能当锤头的面包了，想想就绝望…

> 储存技术的进步真是太棒了！至少现在，我们船上的伙食，只是在风味上比不过饭馆，总体还是健康、新鲜又营养的。

> 更换冰元素瓶和冰雾花的麻烦，比起切掉食材腐烂的部分，也不值一提。

> 以前我还不理解，北斗大姐头每到一个地方，都要专门安排人去询问各种前沿商品的技术，不惜花重金也要搬到船上来用…

> 现在看来，大姐头的长远眼光绝对准狠，任何事情都可以相信她！
