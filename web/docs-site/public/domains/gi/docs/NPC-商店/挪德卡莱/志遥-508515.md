# 志遥

## 基本信息

**位置**: 死兆星级二号舰
**性别**: 男

## 对话内容

**【开场白】**

> 哟！北斗大姐头的好姐妹/哥们，欢迎来到「南十字」船队的二号舰，请将这里当成自家的船！

> 嗯，严格来说，这艘船是「死兆星」号同级的二号舰才对，船队里有许多船龄远大于她的姐姐。

> 不过，本着便于记忆和管理的原则，大姐头定了个规矩——船队里的编号，需要按吨位从大到小排序。

> 所以她在加入船队后，就顺理成章拿到了二号舰的位置，成了许多弟兄们努力转调的目标。

> 当然，荣誉是责任的衍生品。之后，二号舰就会一直在最凶险的海域巡逻，处理那些海盗水匪和魔物，保证航道安全。
