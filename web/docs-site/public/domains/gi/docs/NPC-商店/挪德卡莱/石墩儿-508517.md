# 石墩儿

## 基本信息

**位置**: 死兆星级二号舰
**性别**: 男

## 对话内容

**【开场白】**

> 这不是旅行者吗？我可真走运，刚来甲板上透气，就遇上了你…

> 嗯，我是这艘船的轮机长，通俗点说，就是动力装置的负责人，所以一般闷在船舱里。

> 划桨？哈哈！看来工匠们的小巧思不仅骗过了外人，连自己人都被忽悠了，确实有效！

> 其实，「死兆星」级侧舷的桨是辅助性的部件，推动船体前进的主力，是由机关驱动的水上明轮和水下螺桨。

> 如果遇上了龙蜥之类灵巧的魔物，追不上，也逃不掉，可就尴尬了，只有明轮和螺桨带来的动力才能保证足够的航速。

> 侧舷的船桨其实也可直接连接动力机关，一般在海况极端恶劣时启动，用于提供力矩，辅助稳定船体，或是快速转向。

> 比如许久之前，迎战「跋掣」时，狂浪大作，「死兆星」号就按特别的规律差分驱动左右两舷的船桨，有效减轻了船体的横摇。

> 另外，如果有小型魔物接近船体的盲区，这些大力横挥的船桨也能拍击它们，避免它们贴上水线，爬上甲板。

> 当然啊，船帆还是有用的，在狂风大作的外海，充分运用风能，可以节省不少开销！
