# 梅洛彼得堡

独立于枫丹的司法体系之外，以自治形式存在的水下要塞。是容留受审判而遭放逐的囚人的场所，也是枫丹最大的发条机关生产机构。

| 项目 | 内容 |

## 组织特性

| 特性 | 描述 |
|------|------|
| **互助会** | 约在400年前成立，鼓励弱势罪犯团结互助的团体。 |
| **檐帽会** | 自称为迷茫或孤独的罪人提供帮助的类似同好协会的团体。 |
| **特许食堂** | 梅洛彼得堡中提供食物的地方。 |
| **拳力斗技场** | 举办「拳力斗技」的地方。 |
